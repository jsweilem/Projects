How To Utilize the Analysis Tool


1. Follow the link at the bottom of the page to obtain the JSON packaged data to insert into the Analyzer
2. There are no initial GroupMe's to analyze in the tool, so you need to perform an Add GroupMe operation (explained below)
3. For each of the options, certain fields may or may not be required, so follow the table to know what information is needed for an operation


Options Table Summary
Option GroupMe ID GroupMe JSON Format of Message Body
Add GroupMe Required Required ID: Group_ID
JSON Body: {“ID”:(groupmemessage.json)}
Modify GroupMe Required Required ID: Group_ID
JSON Body: {“ID”:(groupmemessage.json)}
Delete GroupMe Required Not Required ID: Group_ID
Check Stats Required Not Required ID: Group_ID
Find User Required Not Required ID: Group_ID~User
Search Phrase Required Not Required ID: Group_ID~Phrase
Count Phrase Required Not Required ID: Group_ID~Phrase
